## 운영체제 (OS)

### 1. 운영체제(Operating System)
**운영체제**는 컴퓨터의 하드웨어와 사용자를 연결해주며 컴퓨터 자원을 효율적으로 관리하는 기본 소프트웨어이다.
운영체제가 없다면 프로그램을 실행할 수 없고 키보드, 마우스 등의 장비도 사용할 수 없다.

<br/>

-----------------

<br/>

### **2. 운영체제의 주요 역할**
- **프로세스 관리**
	- 여러 프로그램이 동시에 실행되도록 관리한다.
	- CPU의 사용 순서를 결정한다.
	 
- **메모리 관리**
	- 프로그램 간의 충돌을 방지한다.
	- RAM을 효율적으로 분배한다.

- **파일 시스템 관리**
	- 파일 저장, 읽기, 삭제를 관리한다.
	- 폴더 구조를 제공한다.

- **입출력 장치 관리**
	- 컴퓨터, 마우스, 프린터, 디스크 등을 제어한다.
	- 하드웨어를 직접 다루지 않아도 된다.

- **사용자 인터페이스 제공**
	- GUI(그래픽) 또는 CLI(명령어)를 제공한다.
	- 사용자가 컴퓨터를 쉽게 사용할 수 있도록 한다.

<br/>

-----------------

<br/>

### **3. 대표적인 운영체제**
- `Windows` : 개인용 PC
- `macOS` : 애플 컴퓨터
- `Linux` : 서버, 개발, 오픈소스
- `Android` : 스마트폰 운영체제

<br/>

-----------------

<br/>

### **4. 커널 (Kernel)**
**커널**이란 운영체제의 핵심으로 CPU, RAM, 하드웨어를 직접 제어한다.
그렇기에 사용자는 커널을 직접적으로 만질 수 없다.

<br/>

-----------------

<br/>

### **5. 쉘 (Shell)**
**쉘**이란 사용자의 명령을 받아 커널에게 전달하는 프로그램이다.
시스템과 사용자 사이의 인터페이스를 담당하며 명령어 해석기라고 한다.

<br/>
