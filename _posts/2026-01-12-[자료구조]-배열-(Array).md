## Array

### **1. 배열 (Array)**

여러 개의 같은 타입 데이터를 연속된 메모리 구조에 저장하는 자료구조이다.

#### **예시**
```cpp
int arr[5] = {10, 20, 30, 40, 50};
```

-   `arr` : 배열 이름
    
-   `5` : 배열 크기 (원소 개수)
    
-   `{}` : 초기값 (비어있는 상태로 작성 시 모두 0으로 초기화 된다.)

<br/>

-----------------

<br/>

### **2. 배열의 특징**
1. **인덱싱**
	- 인덱스를 통해 배열의 요소에 접근할 수 있다. (EX. ```cout << arr[3];```)
	- 인덱스는 항상 0부터 시작한다.

2. **연속적인 메모리**
	- 연속적인 메모리에 데이터가 저장되기 때문에 원하는 요소에 $O(1)$ 시간에 접근할 수 있다.

3. **크기 고정**
	- 배열의 크기는 선언 시 결정된다.
	- 런타임 시 크기를 변경할 수 없다.

<br/>

-----------------

<br/>

### **3. 배열의 장단점**
1. **장점**
	- 구조가 단순하고 사용하기 쉽다.
	- 인덱스로 통해 각 요소에 $O(1)$ 시간에 접근할 수 있다.

2. **단점**
	- 크기가 고정되어 있다.
	- 삽입 및 삭제 시 요소들을 다시 채워줘야 하기 때문에 느리다.
	- 메모리 낭비가 있다.
		- EX. 배열의 크기는 100이지만 30만 사용하는 경우

<br/>

-----------------

<br/>

### **4. 사용 예시**
```cpp
int arr[3]; 	// 선언
int arr[3] = {1, 2, 3};	 // 선언 및 초기화
int arr[5] = {1, 2};	 // 나머지는 0으로 자동 초기화

// 배열의 전체 요소 접근
for(int i = 0; i < 5; i++)
{
    cout << arr[i] << " ";
}

// 2차원 배열
int arr[2][3] = { {1, 2, 3}, {4, 5, 6} };
```
