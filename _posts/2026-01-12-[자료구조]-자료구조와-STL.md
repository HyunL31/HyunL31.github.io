### **2. STL (Standard Template Library)**
STL이란 C++에서 제공하는 표준 자료구조, 알고리즘 라이브러리이다.

#### **STL 구성요소**
STL은 세 가지 구성요소로 이루어져 있다.

- **Container**: 데이터 저장
	- vector, stack, queue 등
<br/>

- **Algorithm**: 탐색, 정렬 등 알고리즘
	- sort(), find() 등
<br/>

- **Iterator**: 컨테이너 접근(반복자)
	- 포인터처럼 컨테이너 내부를 가리킨다.

```cpp
for(auto it = v.begin(); it != v.end(); it++)
{
    cout << *it << " ";
}

for(int x : v) cout << x;
```
